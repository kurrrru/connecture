  • Name

    サイトの名前であるConnectureは「日本各地の伝統文化（culture）を未来（future）の世代につなぐ（connect）」という意味が込められています。

  • Mission

    過疎化や高齢化、コロナ禍などにより伝統文化の継承が難しくなっている/難しくなりつつある地域における現役世代と次世代の間のギャップ・よく名前が上がる文化から、そうでない文化まで、広く掲載することによって認知度のギャップを埋めるべく作成いたしました。このサイトが、衰退傾向にある伝統文化を多くの方々に、特に若い世代に知っていただくことに貢献できれば幸いです。
    また、伝統文化に現在携わっている方々に利用していただくことで、その文化が消滅してしまう前に記録を残すという役割も兼ね備えております。

  • Development

    現在は日本舞踊のみ掲載しておりますが、今後、食文化や和服文化等のほかの伝統文化にも拡張していく予定です。
